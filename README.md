# While Loop Number Validation Program

This is a beginner Python program that asks the user to enter a number between 1 and 10.
If the number is outside the range, the program keeps asking until a valid number is entered.

## Concepts Used
- while loop
- conditional statements
- user input
- basic validation

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python while_loop_number.py
